## A second-level heading

Body text under H2. Headings should be reachable via anchor links — hover over the heading above and an `#` should appear next to it.

### A third-level heading

Lists work normally:

- First item
- Second item with **bold** and *italic*
- Third item with [a link to xysq.ai](https://xysq.ai)

Ordered too:

1. Step one
2. Step two
3. Step three

> A blockquote. Use these for pull quotes or short citations. They should look distinctive — left border, slightly larger, italic.

## Code

Inline `code` should look distinct from body text. Fenced code blocks should be syntax-highlighted at build time:

```js
// JavaScript
function greet(name) {
  return `Hello, ${name}!`
}
```

```py
# Python
def greet(name: str) -> str:
    return f"Hello, {name}!"
```

```bash
# Shell
curl https://xysq.ai/api/health
```
